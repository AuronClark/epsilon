Parameters
msgThe message to use, such as Error::Memory.
codeThe error code presumably returned by the operation.
overrideWhether to use the given code, if this is true then an exception will always be thrown and code will be ignored as well as omitted from the exception message.
Note
If override is false, an exception will be thrown iff code != 0 (this is also consistent with CL_SUCCESS).
